# Mazmorras y Dragones 2D (terminal)

Juego sencillo estilo roguelike hecho desde cero en Python (sin dependencias externas).

## Requisitos

- Python 3.10 o superior.

## Cómo jugar

```bash
cd dnd_2d_terminal
python3 game.py
```

## Controles

- `w`, `a`, `s`, `d`: moverte.
- `p`: beber poción.
- `i`: ver resumen rápido de enemigos vivos.
- `q`: salir de la partida.

## Objetivo

- Avanza por 3 niveles de mazmorra.
- Sobrevive, sube de nivel y derrota al Dragón final.

## Símbolos del mapa

- `@` héroe
- `E` enemigo normal
- `D` dragón
- `#` muro
- `>` escaleras al siguiente piso
- `$` oro
- `!` poción
